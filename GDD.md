# Part 1: 게임 비전 & 개요

## 1.1 프로젝트 정보

| 항목 | 내용 |
|------|------|
| **제목** | Dev Survivor |
| **장르** | 2D 캐주얼 서바이벌 (뱀서라이크) |
| **플랫폼** | 웹 브라우저 (HTML/CSS/JS) |
| **1판 플레이타임** | 3분 30초 |
| **개발 형태** | AI 협업 프로토타입 실험 |
| **배포** | GitHub Pages — https://rafeeky.github.io/DevSurvivor/ |

## 1.2 한 줄 정의

> AI가 점점 진화하는 환경 속에서, 플레이어는 개발자가 되어 3분 30초 동안 살아남고 오늘의 작업을 끝내야 한다.

## 1.3 핵심 감정 리듬

| 구간 | 감정 |
|------|------|
| 초반 | "할 만한데?" |
| 중반 | "이제 좀 귀찮고 빡세네" |
| 후반 | "이건 진짜 나를 대체할 수도 있겠다" |
| 결말 | "오늘은 내가 이겼나, AI가 이겼나" |

## 1.4 개발 목적

- 링크 공유용 HTML 프로토타입 — 누구나 바로 플레이 가능
- 짧은 기간 안에 코어 루프 검증
- Claude Code + 서브 에이전트를 활용한 AI 협업 실험

## 1.5 성공 기준

### 게임 측면
- 3분 30초 루프가 끊기지 않고 돌아간다
- 1판 종료 후 다시 플레이할 이유가 있다 (메타 성장)
- 처음 보는 사람도 컨셉을 30초 안에 이해한다
- 적의 진화가 설명 없이 직관적으로 읽힌다

### 실험 측면
- AI를 어디에 쓰면 효율적인지 체감한다
- 사람이 직접 붙들어야 하는 판단 지점이 보인다

# Part 2: 코어 루프 & 게임플레이

## 2.1 전체 게임 루프

```
로비 (닉네임 설정 / 캐릭터 선택)
 └─ 시작하기
     └─ 플레이 (3분 30초)
         ├─ 한 판 루프: 적 처치 → 경험치 → 레벨업 선택 → 스킬 강화 → 생존
         └─ 종료 (생존 or 사망)
             └─ 결과 정산
                 └─ 출시 포인트 획득
                     └─ 메타 업그레이드 (선택)
                         └─ 로비 복귀 → 재도전
```

## 2.2 조작 방식

| 입력 | 동작 |
|------|------|
| **방향키** / WASD | 이동 |
| **Q / W / E / R** | 스킬 슬롯 1~4 발동 |
| **ESC** | 일시정지 / 재개 |
| (모바일) 가상 조이스틱 | 이동 |

> **자동 공격 없음**: 모든 전투는 스킬 발동으로 처리한다.
> 기본 이동과 스킬 타이밍이 핵심 플레이.

## 2.3 3분 30초 타임라인

| 구간 | 시간 | 등장 적 | 플레이어 목표 | 감정 |
|------|------|---------|-------------|------|
| 1 | 0:00 ~ 0:30 | 박스봇 | 조작 파악, 기본 생존 | "귀엽고 할 만하다" |
| 2 | 0:30 ~ 1:00 | 박스봇 + 블루패치 | 첫 압박, 첫 레벨업 | "좀 빨라졌네" |
| 3 | 1:00 ~ 1:40 | 블루패치 + 모니터헤드 | 공간 관리 전환 | "이제 귀찮아지네" |
| 4 | 1:40 ~ 2:10 | 블루패치 + 모니터헤드 혼합 | 빌드 체감, 실수 시 위기 | "버티기 쉽지 않다" |
| 5 | 2:10 ~ 2:35 | 제럴드 첫 등장 | 대체 감각 제시 | "저건 나 같은데?" |
| 6 | 2:35 ~ 2:50 | 제럴드 + 잔존 적 | 보스 전 최종 압박 | "마지막이 온다" |
| 7 | 2:50 ~ 3:30 | **미러워커** 등장 | 최종 보스 생존 or 처치 | "결국 완성형이 나왔다" |

## 2.4 보스 처치 플로우

미러워커(보스)를 처치하면:

1. 골드 다이아(최고 등급) 드롭
2. 말풍선 "퇴근하세요! 🎉" 표시
3. 플레이어가 골드를 픽업하면 **Dev Win** 화면 전환

> 타이머 종료 전에 보스를 처치해도 승리 가능.
> 보스를 처치하지 못하고 타이머 종료 시 생존 기준으로 결과 정산.

## 2.5 레벨업 타이밍 (권장)

| 레벨 | 권장 시점 |
|------|---------|
| 1 → 2 | 0:35 ~ 0:50 |
| 2 → 3 | 1:05 ~ 1:20 |
| 3 → 4 | 1:40 ~ 1:55 |
| 4 → 5 | 2:10 ~ 2:25 |
| 5 → 6 | 2:35 ~ 2:50 |

# Part 3: 플레이어 시스템

## 3.1 기본 스탯 (레벨 1 기준, 공통)

| 스탯 | 기본값 | 설명 |
|------|--------|------|
| **최대 HP** | 100 | 0이 되면 사망 |
| **이동 속도** | 220 px/s | 기본 이동 속도 |
| **피격 무적 시간** | 0.4초 | 피격 후 짧은 무적 |
| **스킬 슬롯** | 4개 | Q / W / E / R |

> 캐릭터별 고유 패시브가 플레이 방향을 만든다. 기본 스탯 자체는 동일.

## 3.2 체력 관리

- 기본 HP 회복 없음 (스킬로만 회복)
- HP 30% 이하 시 화면 테두리 붉은 점멸
- 피격 시 전체 화면 빨간 플래시 (0.35 알파, 빠르게 소멸)

## 3.3 닉네임 시스템

- 로비에서 처음 시작 시 닉네임 입력 (한글·영어, 최대 12자)
- localStorage에 저장, 게임 내 HUD 우상단에 표시
- 로비 하단 `[ 닉네임 변경 ]` 버튼으로 언제든 변경 가능

## 3.4 캐릭터 구성

Dev Survivor는 **3종 캐릭터**를 제공하며, 각각 고유 패시브 1개로 플레이 방식을 차별화한다.

### 캐릭터 목록

| 캐릭터 | 역할 타입 | 패시브명 | 해금 조건 |
|--------|---------|---------|---------|
| 개발자 (남) / Adam | **속도형** | 빠른 적응 | 기본 해금 |
| 개발자 (여) / Amelia | **버티기형** | 오래 버티기 | 기본 해금 |
| 뱀파이어 개발자 / Vampir | **야행성형** | *(미정)* | 보스 처치 후 골드 다이아 1회 획득 시 해금 |

---

## 3.5 캐릭터별 고유 패시브

### 개발자 (남) — 속도형 / 빠른 적응

**컨셉**: 처음엔 약하지만 배우는 속도가 빠르다. 경험치를 더 효율적으로 먹고, 첫 성장이 빨리 온다.

**효과**
- 경험치 획득량 +20%
- 첫 레벨업 필요 경험치 감소
- 이동속도 +20 px/s

**플레이 차이**: 초반 빌드 형성이 가장 빠름. 박스봇/블루패치 구간에서 강함.

---

### 개발자 (여) — 버티기형 / 오래 버티기

**컨셉**: 압박이 와도 무너지지 않고 버틴다. 회복, 보호막, 저체력 완화에 강점.

**효과**
- 회복 스킬 회복량 +25%
- 보호막 효율 증가
- 체력 30% 이하 시 받는 피해 -20%

**플레이 차이**: 실수 복구가 쉬움. 초보자 친화적. 제럴드·미러워커 구간 안정성 높음.

---

### 뱀파이어 개발자 — 야행성형 (잠금)

**해금 조건**: 보스 미러워커 처치 후 골드 다이아를 1회 획득하면 자동 해금.

**컨셉**: 야근을 즐기는 타입. (상세 패시브는 추후 확정)

---

## 3.6 캐릭터 선택 화면 표시 항목

- 캐릭터 픽셀 스프라이트 (idle 애니메이션)
- 이름
- 역할 타입 배지 (속도형 / 버티기형 등)
- 패시브명 + 한 줄 설명
- 잠금 캐릭터: 잠금 아이콘 + 해금 조건 안내

# Part 4: 적 시스템

## 4.1 적 진화 계보

```
박스봇 → 블루패치 → 모니터헤드 → 제럴드 → 미러워커
(허술)    (반응)     (방해)       (대체)     (완성)
```

> 서로 다른 종족이 아니라 **하나의 AI가 발전하는 과정**으로 읽혀야 한다.

## 4.2 적 스탯 표

| 이름 | HP | 이동 속도 | 충돌 데미지 | 처치 경험치 |
|------|-----|---------|------------|-----------|
| 박스봇 | 30 | 80 px/s | 10 | 5 |
| 블루패치 | 55 | 130 px/s | 15 | 10 |
| 모니터헤드 | 80 | 70 px/s | 12 | 15 |
| 제럴드 | 150 | 170 px/s | 25 | 30 |
| 미러워커 (보스) | 600 | 200 px/s | 35 | 200 |

## 4.3 적 행동 패턴 상세

### 박스봇
- 플레이어 방향으로 직진
- 특수 패턴 없음
- 다수 동시 스폰 (군집형)

### 블루패치
- 플레이어 추적
- 3초마다 짧은 돌진 (속도 2배, 0.6초간)
- 방향 전환 빠름

### 모니터헤드
- 플레이어와 거리 유지 (150~250px)
- 2.5초마다 **오류탄** 발사 (느린 투사체, 명중 시 이동속도 -30%, 2초)
- 6초마다 **방해 구역** 생성 (반경 80px, 4초간 지속, 진입 시 데미지)

### 제럴드
- 빠른 근접 추적
- 4초마다 연속 돌진 (3회 연속)
- 플레이어 방향 예측 이동

### 미러워커 (최종 보스)
- 고속 추적 (기본)
- 5초마다 강한 돌진 (광역 1회)
- 8초마다 **충격파** (플레이어 중심 200px 원형, 데미지 + 넉백)
- 체력 50% 이하 시 **하위 적 소환** (블루패치 2마리, 20초마다)
- 처치 시 골드 다이아 드롭 → "퇴근하세요!" 연출

## 4.4 스폰 규칙

| 구간 | 스폰 간격 | 동시 최대 적 수 |
|------|---------|--------------|
| 0:00 ~ 0:30 | 2.5초 | 5 |
| 0:30 ~ 1:00 | 2.0초 | 8 |
| 1:00 ~ 1:40 | 1.8초 | 10 |
| 1:40 ~ 2:10 | 1.5초 | 12 |
| 2:10 ~ 2:35 | 2.0초 (제럴드 포함) | 8 |
| 2:35 ~ 2:50 | 2.0초 | 10 |
| 2:50 ~ 3:30 | 미러워커 단독 등장 (나머지 스폰 중지) | — |

- 스폰 위치: 화면 바깥 랜덤 방향에서 등장
- 동시 최대 적 수 초과 시 스폰 대기

# Part 5: 스킬 & 판 내 성장

## 5.1 경험치 & 레벨업

| 레벨 | 필요 경험치 (누적) |
|------|----------------|
| 1 → 2 | 30 |
| 2 → 3 | 80 |
| 3 → 4 | 160 |
| 4 → 5 | 280 |
| 5 → 6 | 450 |

- 레벨업 시 게임 일시 정지 후 **3지선다** 제시
- 선택지: 신규 스킬 획득 / 기존 스킬 강화 (LV.1→LV.2→LV.MAX) / 기본 능력 강화
- 레벨 6 이후에는 스킬 강화 또는 기본 능력 강화만 제공

## 5.2 스킬 목록

### 전투형 스킬

| 스킬 | 역할 | 기본 수치 | 쿨다운 |
|------|------|---------|--------|
| **긴급 패치** | 근거리 범위 공격 | 반경 120px, 데미지 40 | 5초 |
| **디버그** | 단일 고화력 | 가장 가까운 적 1명, 데미지 70 | 2.5초 |
| **고성방가** | 전방 부채꼴 + 넉백 | 부채꼴 120°/160px, 데미지 30, 넉백 100px | 7초 |

### 생존/회복형 스킬

| 스킬 | 역할 | 기본 수치 | 쿨다운 |
|------|------|---------|--------|
| **자동 저장** | 보호막 (피격 1회 무효) | 12초마다 자동 갱신 | 패시브 |
| **커피 한 잔** | 단기 버프 | 이동속도 +40%, 쿨다운 -20%, 5초 지속 | 20초 |
| **산책하기** | 이동속도 강화 | 이동속도 +30% 지속 | 패시브 |
| **멘탈관리** | 피해 감소 + 넉백 | 받는 피해 -40%, 주변 150px 넉백, 4초 지속 | 12초 |
| **쓰다듬기** | 회복 | 최대 HP의 25% 회복 | 20초 |
| **낮잠자기** | 고위험 고회복 | 1.5초 정지 후 HP 50% 회복, 모든 쿨다운 -3초 | 30초 |

## 5.3 스킬 강화 — 3단계 시스템 (LV.1 / LV.2 / LV.MAX)

모든 액티브 스킬은 최대 3단계(LV.MAX)까지 강화 가능하다.
레벨업 3지선다에서 "강화" 선택지를 고를 때마다 한 단계씩 올라간다.

| 스킬 | LV.1 (기본) | LV.2 강화 | LV.MAX 강화 |
|------|------------|---------|------------|
| **긴급 패치** | 데미지 40, 반경 120px, 5초 쿨 | 데미지 +20 | 반경 +40px, 쿨다운 3초 |
| **디버그** | 데미지 70, 단일 1명, 2.5초 쿨 | 데미지 +30 | 관통 2명, 쿨다운 1.5초 |
| **고성방가** | 데미지 30, 넉백 100px, 7초 쿨 | 데미지 +15 | 범위 확장, 쿨다운 4초 |
| **커피 한 잔** | 이속+40%, 5초 지속, 20초 쿨 | 지속 +3초 | 버프 효과 +10%, 쿨다운 -3초 |
| **멘탈관리** | 피해-40%, 4초 지속, 12초 쿨 | 넉백 강화 | 지속 +2초, 쿨다운 -3초 |
| **쓰다듬기** | HP 25% 회복, 20초 쿨 | 회복량 +10% | 회복 중 피해-20%, 쿨다운 -5초 |
| **낮잠자기** | HP 50% 회복, 30초 쿨 | 회복량 +15% | 쿨다운 -8초 |

> HUD 스킬 슬롯 우하단에 LV.1 / LV.2 / **LV.MAX**(금색) 로 표시한다.

# Part 6: 경제 & 메타 성장

## 6.1 스코어 공식

```
최종 점수 = 생존 점수 + 처치 점수 + 진행률 점수 + 보너스
```

| 항목 | 계산식 |

```
Tong (디렉터)
 └─ brief-generator 스킬 → 작업 지시서 (.claude/briefs/)
```
